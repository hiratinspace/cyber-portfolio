# Windows Security Monitoring Lab

## Objective
Build a Windows lab to collect and analyze security telemetry using Sysmon and Event Viewer.

## Environment
- Windows 11 (VMware)
- Sysmon
- Event Viewer

## Data Sources
- Windows Security Log
- Sysmon Operational Log

## Outcomes
- Collected authentication events
- Observed process creation and termination
- Investigated suspicious behavior

## Key Events Observed
- Failed login attempts (Event ID 4625)
- Successful logons (Event ID 4624)
- Process creation events (Sysmon Event ID 1)

## Skills Practiced
- Log analysis
- Event correlation
- Basic incident investigation
