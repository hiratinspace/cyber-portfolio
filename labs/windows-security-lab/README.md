# Windows Security Monitoring Lab

## Objective
Set up a Windows virtual machine to collect and analyze security logs using Sysmon and Event Viewer.

## Environment
- Windows 11 (VMware)
- Sysmon
- Event Viewer

## Data Sources
- Windows Security Log
- Sysmon Operational Log

## Outcomes
- Collected authentication events
- Observed process creation and termination
- Investigated suspicious behavior
